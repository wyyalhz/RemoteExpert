# Remote Expert Skeleton (Qt/qmake, C++11)

## 开发环境：

VMware16.0 + Ubuntu18.04 + Qt5.12.8

使用qmake创建项目

## 目录结构：

remote-expert/
  common/            # 统一协议（protocol.h/.cpp）
  server/            # 服务器（控制台程序）
  home/              # 客户端（Qt Widgets）
  docs/              # 服务器端说明文档

## 使用方法：

用 Qt Creator 打开home.pro 即可，构建并运行客户端
